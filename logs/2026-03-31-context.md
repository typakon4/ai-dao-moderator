## Контракт: AIDAOModerator

**Файл:** `ai_dao_moderator.py`

### Интерфейс

- `__init__(constitution: str)` — конституция DAO
- `submit_proposal(pid, title, body)` — AI через `gl.eq_principle_prompt_comparative` проверяет пропозал на соответствие конституции. Хранит: approved/score/reason
- `vote(pid, voter, support, argument)` — AI скорит аргумент 1-10, балл = вес голоса (не 1 адрес = 1 голос)
- `get_result(pid) -> dict` — взвешенные голоса + pass/fail
- `get_proposal(pid) -> dict` — данные пропозала с AI-вердиктом

### Правила реализации

- Хранилище: `TreeMap` (не `dict`/`list`)
- Proper decorators: `@gl.public.write` / `@gl.public.view`
- Return type annotations на всех методах
- Тесты через genlayer-test с mocked LLM responses
